Project Overview

The Smart Irrigation Control System helps farmers optimize water usage by making intelligent predictions about when to activate irrigation pumps. By analyzing data patterns in soil moisture levels and temperature, the system can recommend the optimal timing for irrigation.

Dataset Structure

The application works with data in the following format:

crop moisture temp pump
cotton 750 25 1
cotton 650 30 1
cotton 350 22 0
cotton 200 28 0

Where:

  • crop: Type of crop (e.g., cotton)
  • moisture: Soil moisture level
  • temp: Temperature in Â°C
  • pump: Target variable (1 = pump ON, 0 = pump OFF)

How to Use

To use this application:

  1. Clone the repository from GitHub
  2. Install the required dependencies
  3. Run the Streamlit application
  4. Upload your dataset or use the provided sample data
  5. Train a model using your preferred algorithm
  6. Make real-time predictions based on current environmental conditions

For detailed installation and usage instructions, please refer to the README on GitHub.

Important Note

This is a Streamlit Python application that cannot run directly on GitHub Pages. To use this application, you'll need to download the source code and run it locally or deploy it to a Python-compatible hosting service like Streamlit Cloud, Heroku, or similar platforms.
